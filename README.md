# SQL Sales Analysis - Chinook Database

# Project Description

This project analyses sales data from the ficticious "Chinook Database"

# Technologies

- Database: PostgreSQL
- SQL GUI: pgAdmin 4
- Data source: Chinook Database ([Github link] https://github.com/lerocha/chinook-database)
